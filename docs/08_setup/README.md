# Setup Guides

This directory contains setup and configuration guides for Buzzer Network.

## Available Guides

### Database Setup
- **DATABASE_CONNECTION_GUIDE.md** - Guide for connecting to various database providers (Supabase, Neon, AWS RDS, etc.)
- **QUICK_DB_SETUP.md** - Quick 5-minute setup guide for Supabase

### Infrastructure Setup
- **RAILWAY_QUICK_SETUP.md** - Quick Railway setup guide
- **REDIS_SETUP.md** - Redis/Upstash cache setup

### Frontend Setup
- **FRONTEND_DEPENDENCIES.md** - Frontend dependency setup

### Security & Keys
- **HOW_TO_GET_PRIVATE_KEY.md** - Guide for obtaining wallet private keys

### Project Setup
- **SETUP.md** - General project setup instructions

## Quick Start

For the fastest setup, see **QUICK_DB_SETUP.md** for Supabase setup.

For production deployments, see **DATABASE_CONNECTION_GUIDE.md** for cloud database options.

For Railway deployment, see **RAILWAY_QUICK_SETUP.md**.
