# Deployment Documentation

This directory contains deployment guides and status documents for Buzzer Network.

## Deployment Guides

### Backend Deployment
- **BACKEND_DEPLOYMENT.md** - Complete backend deployment guide (Railway, Render, Fly.io)
- **QUICK_DEPLOY_BACKEND.md** - Quick deployment guide for Railway

### Frontend Deployment
- **VERCEL_DEPLOYMENT.md** - Vercel deployment guide
- **DELETE_VERCEL.md** - Guide for removing Vercel deployment

### Smart Contracts
- **CONTRACT_DEPLOYMENT.md** - Smart contract deployment guide
- **DEPLOY_CONTRACTS_NOW.md** - Step-by-step contract deployment instructions

## Quick Start

For quick backend deployment, see **QUICK_DEPLOY_BACKEND.md**.

For contract deployment, see **DEPLOY_CONTRACTS_NOW.md**.

## Deployment Checklist

- [ ] Backend deployed to Railway
- [ ] Database connected (Supabase)
- [ ] Environment variables configured
- [ ] Frontend deployed to Vercel
- [ ] Smart contracts deployed (BASE Sepolia/Mainnet)
- [ ] Contract addresses configured in backend
- [ ] Health checks passing
- [ ] API endpoints tested
